## 請解釋後端與前端的差異。
* 前端：使用者真正看到的網頁，瀏覽器透過 html、css、javascript 的原始程式碼轉換後顯示出來
* 後端：遠端的主機，儲存與運算資料，能動態的與資料庫檢索符合的資料

## 假設我今天去 Google 首頁搜尋框打上：JavaScript 並且按下 Enter，請說出從這一刻開始到我看到搜尋結果為止發生在背後的事情。
1. 在瀏覽器輸入 JavaScript 並且按下 Enter
2. 瀏覽器透過電腦發出 Request 到 DNS，詢問 Google 伺服器的 IP 位置並回傳
3. 我的電腦會連接到對應的 IP 位置發出 Request，要求伺服器在資料庫檢索資料
4. 伺服器收到資料後會傳回到我的電腦
5. 瀏覽器透過程式碼轉換使用者看到的網頁
