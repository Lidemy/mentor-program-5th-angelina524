## 跟你朋友介紹 Git

### Git 的基本概念
Git 是一種版本控制的程式，可儲存不同的版本記錄，便於管理。

### Git 的基礎的使用
1. `add`：將檔案加入版本控制，放到暫存區
2. `commit`：建立版本，將暫存區的內容移到倉庫
3. `push`：將本地的資料傳到遠端，例如：可連接 GitHub，遠端儲存 Git 各版本的地方
4. `pull`：將遠端的資料抓下來到本地
